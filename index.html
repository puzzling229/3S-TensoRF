Abstract

We propose Sparse Object Labeled Tensorial Radiance Field (SOL-TensoRF), a method for novel view synthesis together with 3D-consistent object segmentation. Previous methods rely on dense 2D segmentation annotations and suffer from severe marginal performance loss with sparse object segmentation label maps. To address this problem, our method leverages distilled semantic features extracted from input RGB images. We distill these features using a pre-trained image feature extractor as a teacher network, allowing us to obtain object segmentations with sparse labels while maintaining high performance. In addition, as semantic and object labelling is highly correlated with geometry, signed distance fields (SDF) representation is used as geometric representation for better reconstruction result. Furthermore, once trained by sparse object labels, our model can achieve realistic scene editing by manipulating objects in combination with diffusion-based 2D inpainting for mitigating artifact issues. Experimental results show that our model can successfully perform high-quality geometry, appearance and object field reconstruction under sparse segmentation supervision with significantly reduced training time. Additionally, we present high fidelity renderings for object-level editing.

Methods

Given posed RGB images I of a scene with corresponding segmentation annotations S, we aim to build a volumetric scene representation that models appearance, geometry and 3D object segmentations, while using object labels from as few viewpoints as possible. Our proposed method, SOL-TensoRF, enables generation of RGB images and object segmentations from novel viewpoints while preserving the rendered 2D label of the same 3D object across views.
